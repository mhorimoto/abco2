# Debug Messages

関数などの処理単位へのEntry/Exitについてはその判定を容易にするために数値化する。
基本的な考え方は、Enteryで"+"、Exitで"-" 処理単位には各々ユニークな２のべき乗数値だけを与える。足して引くと０になるので、最終的に一回りできると０になるはずである。
